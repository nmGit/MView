mView.dataChestWrapper module¶

class mView.dataChestWrapper.dataChestWrapper(device)[source]¶

The dataChestWrapper class handles all datalogging. An instance of dataChestWrapper should be created in the main class in order to begin datalogging.

Initialize the datalogger, if datasets already exist, use them. Otherwise create new ones.
